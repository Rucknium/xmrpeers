`xmrpeers` is an alpha project. Expect breaking changes in future
versions.

## Installation

Install R. Linux users should install the `r-base` and `r-base-dev`
system packages.

You can install the development version of xmrpeers from
[GitHub](https://github.com/) if you have `git` and `curl` installed on
your system with:

``` r
install.packages("remotes")
remotes::install_github("Rucknium/xmrpeers", upgrade = FALSE)
```
